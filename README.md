🎬 Netflix Pakistan Landing Page Clone

A responsive Netflix Pakistan landing page clone built using HTML5 and CSS3.
This project focuses on recreating the UI of Netflix’s homepage with modern layout techniques, responsive design, and clean styling.

📌 Features

🎥 Netflix-style hero section with background image overlay

📺 Video playback inside TV mockups

📱 Fully responsive design (mobile, tablet, desktop)

❓ FAQ section with hover effects

👶 Kids profile section

🧭 Clean navigation bar and footer

🎨 Google Fonts integration (Poppins & Martel Sans)  
🛠️ Technologies Used

HTML5

CSS3

Google Fonts

Flexbox & CSS Grid

Media Queries for responsiveness
